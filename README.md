A desktop tool for OCR capture, clipboard translation, and personal vocabulary management.

Features

- Floating toolbar on startup
- OCR screen capture
- Automatic clipboard detection (Ctrl + C)
- Popup translation window
- Add words directly into dictionary
- Dictionary home interface
- Vocabulary collection system (not yet)
- Editable fields (reading, translation, tags) (not yet)
- Search, tag filtering, pagination (not yet)
